# BirdGame
BirdGame is a simple clone of Flappy Bird developed in Java using Slick2D library.

## Features
- auto pilot
- realistic physics

## Controlls
- spacebar - jump
- escape - exit
- A - toggle auto pilot
- D - toggle debug menu
